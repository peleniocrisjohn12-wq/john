ExerciseS Reflection

Working with inline CSS helped me understand how I can style specific elements directly in the HTML. For example, using style="background-color: yellow;" on a heading or a table cell allowed me to quickly change its appearance. I realized that inline CSS is useful for small, one-time changes, but if I use it too much, the code can become messy. This experience taught me to be careful when using inline styles and to balance design with clean coding.
